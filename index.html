Fitur Validasi

  • Input Teks: Panjang karakter, tipe, dan filter karakter khusus.
  • Elemen Select: Memastikan opsi telah dipilih.
  • Radio: Memastikan minimal satu pilihan terpilih.
  • Input File: Validasi ekstensi dan ukuran file (MB).

Atribut data- untuk Validasi

Atribut Deskripsi
data-required Wajib diisi.
data-alphabetic Hanya huruf (A-Z, a-z).
data-specialchar Karakter khusus yang dilarang.
data-minlength Minimal jumlah karakter.
data-maxlength Maksimal jumlah karakter.
data-email Format email valid.
data-isnumber Hanya angka.
data-extfile Ekstensi file (contoh: jpg,png).
data-size Ukuran maksimal file (MB).
data-nohtml Melarang tag HTML.

Cara Menggunakan

Siapkan form Anda dengan class validate dan atribut data yang diinginkan.

<form id="myForm">
  <div class="form-group">
    <label>Nama</label>
    <input type="text" name="name" class="form-control validate-text vl_name" data-required="true" data-alphabetic="true" />
    <div class="invalid-feedback"></div>
  </div>

  <button type="submit" class="btn btn-primary">Submit</button>
</form>

Metode validasi berbasis objek yang lebih fleksibel tanpa mengotori HTML dengan banyak atribut data.

Tabel Rules Lengkap

Rule / Atribut Tipe Deskripsi Contoh Objek (v4)
required Semua Bidang tidak boleh kosong. ["required"]
alphabetic Text Hanya huruf (A-Z, a-z). ["alphabetic"]
email Text Format email valid. ["email"]
no_html Text Melarang tag HTML (Anti-XSS). ["no_html"]
min:{n} Text Minimal jumlah karakter. ["min:5"]
max:{n} Text Maksimal jumlah karakter. ["max:20"]
ext:{list} File Format file (jpg,png,pdf). ["ext:pdf,docx"]
size:{n} File Ukuran maksimal file (MB). ["size:2"] 
const rules = {
  "#deskripsi": {
    type: "text",
    rules: ["required", "no_html", "min:5"]
  },
  "is_active": {
    type: "radio",
    rules: ["required"]
  }
};

if (!validateFormRules("#form", rules)) {
  return;
}

Handling Error (Backend)

1Validation! mendukung integrasi otomatis dengan pesan error dari Laravel atau CodeIgniter.

// contoh data dari error ajax
const data = jqXHR.responseJSON;

if (data.error_class && data.error_string) {
  handleCodeIgniterError(data);
} else if (data.errors) {
  handleLaravelError(data);
} else if (data.error_class && data.error_string) {
  handleCodeIgniterErrorClass(data);
} else if (data.errors) {
  handleLaravelErrorClass(data);
} else {
  alert('Terjadi kesalahan. Silakan coba lagi.');
}
